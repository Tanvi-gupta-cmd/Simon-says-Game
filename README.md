# Simon Says Game 🎮

A fun and interactive **Simon Says memory game** built using HTML, CSS, and JavaScript. The game challenges players to remember and repeat an increasingly long sequence of colors. It helps improve memory and concentration while providing an engaging user experience.

## 📌 Project Description

Simon Says is a classic memory game where the player must repeat a sequence of colors generated by the game. Each round adds a new color to the sequence, making it progressively harder. If the player clicks the wrong color, the game ends and the score is displayed.

## ✨ Features

* Interactive color buttons with animations
* Increasing difficulty with every level
* Sound and visual feedback
* Score tracking
* Game reset after losing
* Responsive design for different screen sizes

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation)

## 🎯 How to Play

1. Press any key or click **Start** to begin the game.
2. The game will highlight a color in the sequence.
3. Remember the sequence and repeat it by clicking the buttons in the same order.
4. Each level adds a new step to the sequence.
5. If you click the wrong color, the game ends.
